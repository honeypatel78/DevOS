# Angular + Node.js + MSSQL Project

## 📁 Project Structure
- `frontend/` — Angular frontend
- `backend/` — Node.js backend
- MS SQL used as database

## 🚀 How to Run

### Backend:
1. `cd backend`
2. `npm install`
3. Create `.env` with your DB credentials
4. `node server.js`

### Frontend:
1. `cd frontend`
2. `npm install`
3. `ng serve`

## 🛠 Technologies
- Angular CLI
- Node.js (Express)
- Microsoft SQL Server
